## АННОТАЦИЯ

Данный документ представляет собой руководство оператора для работы с веб-приложением «Фамильный Круг — сервис совместного создания и визуализации семейного древа». Руководство предназначено для конечных пользователей системы и содержит описание функциональных возможностей приложения, требований к техническому и программному обеспечению, порядок запуска и эксплуатации приложения, а также перечень возможных сообщений, отображаемых системой, и рекомендации по их обработке.

---

## 1. НАЗНАЧЕНИЕ ПРОГРАММЫ

### 1.1. Функциональное назначение

Приложение «Фамильный Круг» предназначено для создания, хранения и совместного редактирования интерактивных семейных деревьев. Основной функционал включает:

- построение генеалогического дерева с визуализацией связей между членами семьи;
- хранение биографической информации о родственниках (даты, места рождения/смерти, фотографии, документы);
- поддержку совместного редактирования дерева несколькими пользователями с разграничением прав доступа;
- импорт и экспорт данных в стандартных генеалогических форматах (GEDCOM, JSON, PDF, PNG, SVG);
- просмотр событий жизни родственников в хронологическом порядке.

### 1.2. Эксплуатационное назначение

Приложение является веб-сервисом и предназначено для использования в домашних условиях, а также в профессиональной деятельности генеалогов и историков семей. Доступ к приложению осуществляется через стандартный браузер без необходимости установки дополнительного программного обеспечения. Предусмотрены два режима участия в дереве: владелец и соучастник с назначенной ролью (редактор, наблюдатель, комментатор).

### 1.3. Состав функций

- **Управление деревьями:** создание, редактирование и удаление семейных деревьев; приглашение участников по электронной почте с назначением роли.
- **Управление людьми:** добавление, редактирование и удаление карточек участников дерева с указанием имени, фамилии, пола, дат и мест рождения/смерти, фотографии и биографических заметок.
- **Управление связями:** установка связей «родитель–ребёнок», «супруги», «опекун», «усыновлённый».
- **Визуализация:** три режима отображения дерева — классическое «Древо», круговой «Веер» и «Хронология».
- **Поиск:** быстрый поиск участника по имени внутри дерева с переходом к его карточке на графе.
- **Путь родства:** определение степени и характера родства между двумя выбранными участниками дерева.
- **Медиа:** загрузка фотографий и документов (PDF, DOC, XLS, TXT) к карточкам персон и к событиям.
- **События:** добавление именованных событий (рождение, брак, переезд и др.) с датой, местом и описанием.
- **Комментарии:** добавление текстовых комментариев к карточкам участников.
- **Статистика:** автоматический расчёт и отображение статистики дерева (число поколений, распределение по полу, возрасту, распространённые фамилии).
- **Экспорт:** выгрузка дерева в форматах PDF, PNG, SVG, GEDCOM (.ged) и JSON.
- **Импорт:** загрузка данных из файла формата GEDCOM с предварительным просмотром содержимого.
- **Журнал изменений:** аудит-лог всех действий участников дерева (добавление, редактирование, удаление связей и людей).
- **Тёмная тема:** переключение интерфейса между светлой и тёмной цветовой схемой.

## 3. ВЫПОЛНЕНИЕ ПРОГРАММЫ

### 3.1. Запуск приложения

Для запуска приложения пользователю необходимо:

1. Открыть установленный веб-браузер.
2. В адресной строке ввести адрес приложения и нажать клавишу **Enter**.
3. Дождаться загрузки страницы входа.

Приложение не требует предварительной установки. При первом посещении система автоматически перенаправляет неавторизованного пользователя на страницу входа.

### 3.2. Описание страницы входа

При открытии приложения неаутентифицированный пользователь видит страницу входа (см. Рисунок 1). Страница разделена на две части:

- **Левая часть** — приветственный блок с описанием возможностей приложения.
- **Правая часть** — форма входа с полями «Email» и «Пароль».

Для входа пользователю необходимо:

1. Ввести адрес электронной почты в поле **Email**.
2. Ввести пароль в поле **Пароль** (для просмотра пароля нажать иконку глаза).
3. Нажать кнопку **Войти**.

При успешной аутентификации система перенаправляет пользователя на страницу «Мои деревья». При вводе неверных данных отображается сообщение об ошибке.

Для перехода к регистрации нового аккаунта следует нажать ссылку **Зарегистрироваться**.

### 3.5. Описание графа семейного дерева

При нажатии кнопки «Открыть» пользователь переходит к графу семейного дерева (см. Рисунок 4) — основному рабочему экрану приложения.

**Верхняя панель инструментов** содержит:

- Стрелку назад и название дерева с меткой роли.
- Поле поиска по участникам.
- Кнопки переключения режима отображения: **Древо**, **Веер**, **Хронология**.
- Кнопку обновления данных.
- Кнопку **Добавить** — добавление нового участника.
- Кнопку **«...» (Ещё)** — дополнительные функции.

**Граф дерева** отображает участников в виде цветных кружков (синий — мужской пол, розовый — женский, серый — не указан). Внутри кружков — инициалы или фотография. Под кружком — имя и год рождения участника.

Управление графом:

- **Клик по кружку** — открытие боковой панели с данными о персоне.
- **Кнопка «+»** над кружком — добавление нового участника в связи с данной персоной.
- **Кнопка «–»** под кружком — свернуть/развернуть ветвь потомков.
- **Кнопки «+» / «–»** в правом нижнем углу — масштабирование графа.
- **Кнопка рамки** — сброс масштаба и центрирование графа.
- **Перетаскивание** графа мышью — перемещение по полотну.

### 3.8. Добавление нового участника дерева

### 3.9. Описание панели персоны

При нажатии на карточку участника в графе открывается боковая панель с подробной информацией (см. Рисунок 8). Панель содержит четыре вкладки:

**Вкладка «Обзор»** — основная биографическая информация о персоне и раздел комментариев. В поле ввода под заголовком «Комментарии» можно написать комментарий и отправить его нажатием **Enter**.

**Вкладка «События»** (см. Рисунок 9) — список жизненных событий участника (рождение, свадьба, переезд и др.). Для добавления события нажать кнопку **+ Добавить**.

**Вкладка «Файлы»** — фотографии и документы, прикреплённые к персоне. Поддерживаются форматы изображений и PDF/DOC/TXT.

**Вкладка «Связи»** — список родственных связей персоны (родители, дети, супруги, опекуны).

Внизу панели расположены кнопки:

## 4. СООБЩЕНИЯ ОПЕРАТОРУ

В процессе работы с приложением пользователю могут отображаться следующие сообщения:

| Сообщение | Причина | Действие оператора |
|---|---|---|
| «Неверный логин или пароль» | Введены некорректные учётные данные | Проверить правильность email и пароля; убедиться, что аккаунт зарегистрирован |
| «Пользователь с таким email уже существует» | Попытка регистрации на уже занятый email | Использовать другой адрес электронной почты или войти в существующий аккаунт |
| «Пароли не совпадают» | Поля «Пароль» и «Подтверждение пароля» содержат разные значения | Повторно ввести пароль в оба поля |
| «Пароль должен содержать минимум 6 символов» | Введён слишком короткий пароль | Выбрать пароль длиннее 5 символов |
| «Имя обязательно» | Не заполнено обязательное поле «Имя» при создании карточки | Ввести имя участника |
| «Нет входящих приглашений» | Текущий пользователь не получал приглашений | Дождаться приглашения от владельца дерева |
| «Нет событий. Добавьте события к людям в дереве» | В режиме «Хронология» у участников не добавлено ни одного события | Перейти в панель персоны → вкладка «События» → добавить событие |
| «Выберите двух людей, чтобы узнать, как они связаны» | Функция «Путь родства» открыта, но участники не выбраны | Выбрать двух участников из выпадающих списков |
| «Импортированные люди и связи будут добавлены к существующим данным дерева» | Информационное сообщение при импорте GEDCOM | Убедиться в правильности выбранного файла перед нажатием «Предпросмотр» |
| Ошибка сети / «Не удалось загрузить данные» | Потеря соединения с сервером | Проверить подключение к интернету и обновить страницу |
| «Дополнительных данных нет» | В карточке персоны не заполнены биографические поля | Нажать «Редактировать» для добавления информации |
| «Нет комментариев» | К персоне ещё не добавлено комментариев | Написать комментарий в поле ввода и нажать Enter |
